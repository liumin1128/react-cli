# react-cli
React-cli基于Webpack3搭建的react开发环境。

使用方法

#### 1.安装依赖

```
cnpm i
```


#### 2.编译动态链接库

```
npm run dll
```

#### 3.启动编辑模式

```
npm start
```

#### 4.打包

```
npm run build
```
